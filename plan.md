## 2. Исправление проблемы с поиском пользователей

В файле `server/routes/users.js` отсутствует обработчик для маршрута `/users/search`. Необходимо добавить этот обработчик, который будет искать пользователей по имени.

## 3. Исправление проблемы с дублированием штрихов при входе в комнату

Проблема в файле `client/src/store/canvasState.js` в обработчике события `drawsReceived`. При получении штрихов с сервера, они добавляются к существующим, что приводит к дублированию. Необходимо очищать существующие штрихи перед добавлением новых.

## 4. Улучшение алгоритма заливки для более ровных краев

Проблема в файле `client/src/tools/Fill.js` и `client/src/services/CanvasService.js`. Текущий алгоритм заливки оставляет пустоты на границах. Необходимо улучшить алгоритм, добавив сглаживание краев и более точное определение границ.
